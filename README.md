# AssemblyGenerator
A console app for generate a AssemblyInfo.cs (_or modify the .NET Core Project version now_) from a GitHub Repository.

### How to use

Windows:

> AssemblyGenerator.exe --repo **<User/Repo>** --major **1** --minor **0**

Linux:

> ./AssemblyGenerator --repo **<User/Repo>** --major **1** --minor **0**
